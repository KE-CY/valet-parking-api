# valet-parking-api

A backend system for managing valet parking operations, including vehicle check-ins, check-outs, staff assignments, and parking slot tracking.


# Env

```
node.js >= 22.17.0
PostgreSQL = 16.4

```

環境變數設定
請依照專案根目錄中的 .env.sample 檔案建立 .env 檔案，並填入相對應的設定值

# Git Flow 流程說明

本專案使用 Git Flow 流程來管理分支，主要包含 `main`和 `develop` 兩個常態分支。

# 專案啟動
使用 Node.js 本機啟動
```bash
npm install
npm run build
npm start
```
